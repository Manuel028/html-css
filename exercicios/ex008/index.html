Codigo-fonte

Para resolver as diferenças de espaços entre as letras num Codigo-fonte usamos a tag <code>

Codigo sem a tab <code>:
O comando document.getElementById('teste') e escrito em linguagem Javascript.

Codigo com a tab <code>:
O comando document.getElementById('teste') e escrito em linguagem Javascript.

Um caso:
quando usamos a tab <code> para descrever o Codigo-fonte, o codigo nao fica organizado, como:

num = int(input('Digite um numero')) if num % 2 == 0: print(f'0 numero{num} e par') else: print(f'0 numero{num} e impar) print('Fim do programa')

Para organizarmos na forma de codigo fonte temos que incluir toda tag <code> no tab <pre> 

         
    num = int(input('Digite um numero'))
    if num % 2 == 0:
        print(f'0 numero{num} e par')
    else:
        print(f'0 numero{num} e impar)
    print('Fim do programa')
