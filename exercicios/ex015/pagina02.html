Estilos Externos sao estilos usados em todo o documento(outras paginas) e a formatacao aqui afeta todos sites do documento. Para isso basta adicionar a tag <link> escolher o CSS.STYLE(criando um arquivo css href"nome e extensao css") e coloca-se ou faz-se toda a formatacao no css, a tag link css adicionado deve estar em todos documentos(sites) html de modo a referir/aplicar os estilos criado no css . Estilos Locais e' um estilo aceitavel em CSS, o recomendavel pois a formatacao e' feita em todo o documento e afeta todas as paginas do documento.

Mas tambem tambem se adicionar os estilos Inline e Internos portanto isso so afetara a linha e documento dum so site.
