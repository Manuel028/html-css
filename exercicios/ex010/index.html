Trabalhando com links

Usando links externos

Para adicionarmos um link externo usamos a tag <a href="o link do site externo" e o texto link>

Voce pode acessar o meu repositorio offline no Youtube

So que tem um problema, quando clica-se no link externo o site do link e' sobreposto ao do site mae, ou seja o link externo vai ser aberto na mesma aba fazendo com que o site mae esteja oculto.
Para isso devemos adicionar o parametro target e rel dentro da tag ficando <a href="o link do site externo" target = "blank" rel = "external">

Agora o link externo vai ser aberto numa outra abb

Voce pode acessar as minhas transferencias offline no Youtube

Usando links internos

Para criar links internos devemos criar um outro site dentro da nosso arquivo e colocarmos dentro da tag(no site mae) <a no href=" o nome do outro site" e tambem o mecanismo next para indicar proxima pagina e o seu texto link.>

Esta e' a primeira pagina do site. Se quiser, pode acessar tambem a minha Segunda pagina

Se quiser pode tambem ver a pagina de noticias: Noticias do Dia

Links para Downloads

Para links de downloads ainda usamos a tag <a> e devemos colocar o arquivo a ser feito o download no nosso arquivo mae e mencionarmos o arquivo na tag <a>, mas assim o downloade e' direcionado para a extensao do arquivo no navegador, para o download direto adicionarmos na tag o parametro download="nome do arquivo"
