Trabalhando com Listas

Listas Ordenadas

Para criar lista Ordenadas usamos a tag <ol> e dentro dela listamos os itens em tag <li>

Vamos listar uma rotina diaria:

  1. Acordar;
  2. Escovar os dentes;
  3. Fazer banho;
  4. Matabichar;
  5. Ir a escola.

Para criar uma lista mais personalizada onde o indice diferente e iniciando em qualquer posicao, adicionamos dentro da tag <ol> type ="" para indicar tipo de indice e o start ="" para a posicao
